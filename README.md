# ADaM仕様書アカデミー

完全素人から仕様書が書ける、ADaM仕様書の教科書。

## 機能

- **ADaM基礎**: CDISC標準の全体像、SDTMとの関係、仕様書の構成要素、用語集
- **データセット詳細**: ADSL/BDS/OCCDS/Otherの4クラスを変数一覧・サンプルデータ付きで解説
- **変数辞書**: 50以上の標準変数をカテゴリ・クラス・キーワードで検索
- **作成ガイド**: SAP確認からレビューまで6ステップのハンズオンガイド
- **仕様書ビルダー**: データセット・変数を選んで仕様書テンプレートを自動生成（CSV出力対応）
- **理解度クイズ**: 12問のクイズでADaM知識を5段階診断

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- レスポンシブ対応（スマホ・PC）
